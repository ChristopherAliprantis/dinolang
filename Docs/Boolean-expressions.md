NOTE: all boolean expressions return a bool 

**==(type, type)**

example: print(==(2, 2));<br>
output: True

**!=(type, type)**

example: print(!=(2, 2));<br>
output: False

**>(num, num)**

example: print(>(3, 8));<br>
output: False

**<(num, num)**

example: print(<(3, 8));<br>
output: True

**or(bool, bool)**

example: print(or(true, false));<br>
output: True

**and(bool, bool)**

example: print(and(false, false));<br>
output: False
