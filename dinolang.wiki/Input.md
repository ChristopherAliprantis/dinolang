**ReadKey(bool);**

it returns the string of the key you entered<br>
(enter key returns NL)

the boolean parameter represents hiding the entered key<br>
false, the key you enter is shown<br>
true, the key you enter is hid<br>

**ReadLineP();**

it's ReadLine(); but it hides the input<br>
like entering a password in a cli.<br>
so it's called ReadLineP();, Read Line Password.<br>
